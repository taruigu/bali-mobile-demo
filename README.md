# 芭利礼品兑换平台 - 移动端项目

## 项目概述
这是芭利礼品兑换平台的移动端优化版本，完全保持与桌面版本的功能一致，但针对移动设备进行了全面优化。

## 项目特点

### 🎨 移动端优化设计
- **响应式布局**：完全为小屏幕设备优化
- **触摸友好**：所有交互元素都符合移动设备操作习惯
- **快速加载**：优化的资源和精简的CSS
- **无缝导航**：汉堡菜单导航，节省屏幕空间

### ✨ 核心功能
- 🏠 **首页**：精选礼品、分类浏览、品牌优势展示
- 🎁 **礼品分类**：按类别浏览所有礼品
- 💼 **企业服务**：企业客户的定制化解决方案
- 👥 **关于我们**：公司介绍、团队展示、发展历程

### 🔧 技术架构
- **HTML5**：语义化标签，移动端兼容
- **CSS3**：Flexbox和Grid布局，CSS变量主题系统
- **Web Components**：自定义元素（navbar、footer）
- **JavaScript**：触摸事件处理、交互反馈

## 关键移动端优化

### 1. 导航栏（Navbar）
- **桌面版**：全部显示导航链接
- **移动版**：汉堡菜单（☰），点击展开/收起

### 2. 网格布局
- **分类网格**：桌面4列 → 移动2列
- **服务网格**：桌面3列 → 移动1列
- **团队网格**：桌面4列 → 移动2列

### 3. 字体尺寸（相比桌面版减小）
- H1: 3rem → 1.8rem
- H2: 2rem → 1.5rem  
- H3: 1.5rem → 1.1rem
- 正文: 1rem → 0.95rem

### 4. 内边距和间距（优化为小屏幕）
- 容器内边距: 20px → 15px
- 卡片间距: 20px → 15px
- 导航栏内边距: 40px → 15px

### 5. 触摸反馈
- 按钮按下时透明度变化（opacity: 0.8）
- 禁用双击缩放（防止意外放大）
- 按钮全宽显示，便于触摸

## 使用指南

### 查看项目
1. 在浏览器中打开 `index.html`
2. 使用开发者工具切换到移动设备模式（Ctrl+Shift+M / Cmd+Shift+M）
3. 测试不同的设备尺寸（320px, 375px, 414px等）

### 修改内容
所有内容通过HTML模板定义，位于各页面的 `<template>` 标签中。
修改模板中的id对应的文本即可更改页面内容。

### 修改样式
- **颜色**：修改CSS变量（style.css顶部）
- **尺寸**：修改对应组件的CSS（style.css中各部分清晰注释）
- **布局**：修改grid/flex属性

### 添加功能
各组件都是Web Components，可扩展：
- 修改对应的js文件添加新功能
- 在Shadow DOM中使用样式以避免全局污染

## 响应式断点

虽然这是移动端优化版本，但也支持基础的响应式设计：

```css
/* 超小屏幕 (320px - 480px) - 默认样式 */
/* 小屏幕 (480px - 768px) - 使用CSS变量根据需要调整 */
/* 平板及以上 - 考虑添加 @media (min-width: 768px) 规则 */
```
